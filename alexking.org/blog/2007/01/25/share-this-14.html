Share This 1.4

January 25, 2007

I’ve updated my Share This plugin to version 1.4. This includes a number of minor changes since the last release:

  • New method of including Prototype for WP 2.0.x users (can be used by other plugins as well) to avoid including Prototype twice, if possible. This technique is also used in my new WP Grins release.
  • StumbleUpon is included as a default link.
  • Updated FAQ in the README, addressing a few issues I’ve seen in the forums and via e-mail.
  • Added a robots="noindex, noarchive" to the Share This page for each post.
  • Added a missing > symbol.
  • Now using wp_mail() instead of the straight PHP mail() function.
  • Explicitly set the z-index for the pop-up form.

I haven’t gotten around to some of the more significant changes I plan to make, but I figured getting these out there sooner rather than later was a good idea.

Also, I’ve seen a number of translations popping up, if someone wanted to head up an effort to get proper translation files included, I’d be happy to include them. All the strings are abstracted already.

The download and more information are available on my WordPress Plugins page.

If you have any trouble with this, please open a thread in the WP Support Forums and send me the link.

This post is part of the following projects: Share Icon, ShareThis. View the project timelines for more context on this post.

    Facebook can be added by doing these simple steps:

    1. In Site Admin, click on Plugins –> Under Share This, click Edit. You should now be editing the share-this/share-this.php file.
    2. I went the easy route and deleted one of the existing bookmarking services. I picked BlinkList because I’ve never heard of it, you can pick your own, or add a new one.
    3. This is what the facebook template looks like:
    )
    , ‘facebook’ => array(
    ‘name’ => ‘Facebook’
    , ‘url’ => ‘http://www.facebook.com/share.php?u={url}&Title={title}’

    4. For the icon, you’ll need to create a gif file. The easiest way to do this is to download http://www.facebook.com/favicon.ico. Rename it to facebook.gif (it has to be that name) and upload it to your share-this plugin folder.

    Enjoy.
