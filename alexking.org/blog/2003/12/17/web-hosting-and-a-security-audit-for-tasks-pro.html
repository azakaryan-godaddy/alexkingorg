Web Hosting and a Security Audit for Tasks Pro

December 17, 2003

In the next few weeks, I’m going to be moving my site to a blazing new server at Austin Dev (www.austindev.com). I mention this now because this past week I was talking with Chris (from Austin Dev) about doing a security audit of Tasks Pro. Tasks Pro is the first product I’ve built that has built-in security (instead of relying on .htaccess/directory security) and I want to make sure I haven’t left any backdoors open before I release it to the public. The problem is that a good security audit is expensive…

Chris is one of my Tasks Pro sponsors and he has very generously offered to knock the price for the security audit down to zero if I can refer enough hosting business to him. This certainly sounds like a fair deal to me.

He is now offering a ‘Personal Plan’ hosting package for only $8/month! This is an ideal plan for someone who wants to set up a personal site and/or a blog (perhaps using WordPress?). He also has great plans for everything from small businesses to your own dedicated server.

If you are thinking about putting up a web site or are looking for a web host, I highly recommend Austin Dev. Their rates are very fair, their servers and connection are fast and they have been very responsive to support inquiries I’ve had.

If you sign up with Austin Dev, be sure to mention my name so the referral will be applied towards the Tasks Pro security audit. I need to refer 15 new regular hosting clients to him or 2 new dedicated server clients in order to get the free security audit. This is a great opportunity for you to get excellent hosting service and make Tasks a better product at the same time.

As a special incentive to make it even easier to make the switch, if you are one of the first 15 people to sign up at Austin Dev with me as a referral, I will help you migrate your files and/or MySQL database to your new account (if you would like the help).
